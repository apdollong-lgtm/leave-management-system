# ระบบลางานออนไลน์ (Google Apps Script + Google Sheets)

โปรเจกต์ตัวอย่างเว็บแอประบบลางานภาษาไทย โดยใช้ **Google Apps Script** เป็น backend และบันทึกข้อมูลใน **Google Sheets**.

## โครงสร้างไฟล์

- `Code.gs` : ฟังก์ชัน backend สำหรับโหลดข้อมูล, บันทึกคำขอลา, อนุมัติ/ปฏิเสธ
- `Index.html` : หน้าเว็บแดชบอร์ด + ฟอร์มยื่นลา + ตารางอนุมัติ
- `appsscript.json` : ตั้งค่าโปรเจกต์ Apps Script

## วิธีใช้งาน

1. สร้าง Google Spreadsheet ใหม่ 1 ไฟล์
2. ไปที่ `Extensions > Apps Script`
3. ลบไฟล์เริ่มต้น แล้วคัดลอกโค้ดจากโปรเจกต์นี้ลงไป
4. ผูก Apps Script กับ Spreadsheet เดิม (container-bound script)
5. กด `Deploy > New deployment > Web app`
   - Execute as: `User deploying`
   - Who has access: `Anyone` (หรือจำกัดตามองค์กร)
6. เปิด URL ของ Web App

## ฟังก์ชันหลัก

- ยื่นคำขอลา (ชื่อ, แผนก, ประเภท, วันที่, เหตุผล)
- แดชบอร์ดสรุป
  - คำขอรออนุมัติ
  - อนุมัติวันนี้
  - พนักงานที่กำลังลา
- ผู้จัดการอนุมัติ/ปฏิเสธคำขอได้จากหน้าเดียวกัน
- ปุ่ม `ใส่ข้อมูลตัวอย่าง` เพื่อ seed ข้อมูลทดสอบ

## หมายเหตุ

- ครั้งแรกระบบจะสร้างชีตชื่อ `LeaveRequests` พร้อมหัวคอลัมน์ให้อัตโนมัติ
- หากต้องการเก็บประวัติผู้อนุมัติเพิ่ม สามารถเพิ่มคอลัมน์ใน `Code.gs` ได้ทันที
